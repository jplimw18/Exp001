#Ex 001

**Receber um vetor qualquer e retornar pares de números que a diferença (absoluta) entre eles sejam as mínimas possíveis**

**in**  [1, 2, 3, 4]    **out** "1 2" "2 3" "3 4"

**in**  [5, -2, 12, 3]    **out** "5 3"

**in**  [1, 2, 9]    **out** "1 2"

**in**  [-1, -5, 1]    **out** "-1 1"

**in**  [0, 19, 13]    **out** "19 13"
